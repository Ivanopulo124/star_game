Basic game application. How to run on linux desktop

1. Install sdl2 and sdl2_gfx packages
2. mkdir build
3. cd build
4. cmake ..
5. make

How to run on Mac OS

1. Install sdl2 and sdl2_gfx packages
2. mkdir build
3. cd build
4. cmake .. -G "Unix Makefiles"
5. make
